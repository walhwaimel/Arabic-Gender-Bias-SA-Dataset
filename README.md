# Arabic Gender Bias Sentiment Dataset

This repository contains a synthetically generated counterfactual dataset for evaluating gender bias in Arabic sentiment analysis models.

## Dataset Description

- 42 manually designed templates (Modern Standard Arabic)
- 20 identity terms (male / female / neutral forms)
- 2,520 total sentences
  - 840 male variants
  - 840 female variants
  - 840 neutral variants

The dataset is fully balanced across gender categories and generated using a controlled template-based approach to isolate gender as the only manipulated variable.
